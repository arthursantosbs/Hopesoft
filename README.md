# HopeSoft

Sistema de frente de caixa e gestao comercial construído com Spring Boot.

## Estrutura

- `src/main/java/com/hopesoft`: codigo principal da aplicacao
- `src/main/java/com/hopesoft/model`: entidades e enums do dominio
- `src/main/resources`: configuracoes e recursos estaticos
- `src/test/java/com/hopesoft`: testes automatizados

## Stack

- Java 21
- Spring Boot 4
- Spring Web MVC
- Spring Data JPA
- Spring Security
- PostgreSQL
- Lombok

## Status atual

O projeto ja esta padronizado com o namespace `com.hopesoft` e com a base inicial dos modelos de dominio.
