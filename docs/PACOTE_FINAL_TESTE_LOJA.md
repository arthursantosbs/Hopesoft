# Pacote Final de Entrega para Teste

## Estado desta entrega

Este pacote deixa o HopeSoft pronto para uma rodada de teste operacional em loja ou ambiente simulado.

## O que esta incluso

- backend Spring Boot no mesmo repositorio
- frontend React no mesmo repositorio
- Docker com PostgreSQL, backend e frontend
- scripts de instalacao e inicializacao para Windows
- manuais de instalacao, operacao e roteiro de teste

## Enderecos

- sistema: `http://localhost:3000`
- backend: `http://localhost:8080`
- healthcheck: `http://localhost:8080/health`
