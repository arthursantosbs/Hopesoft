# 📦 HOPESOFT - Instalação para Clientes

Pré-requisito: **Docker Desktop** instalado e funcionando.

## Passo a passo (Windows)
1. No seu computador, execute `INSTALAR_HOPESOFT.bat` (clique 2x).
2. Ao terminar, **reinicie** o computador (recomendado).
3. Clique 2x no atalho **HopeSoft** criado no Desktop.
4. Aguarde **2 a 5 minutos** na primeira vez.

## Primeiro acesso
No navegador:
`http://localhost:8080/index.html`

## Login padrão
Email: `admin@hopesoft.com`  
Senha: `hopesoft123`

## Se der erro
- Docker não encontrado/parecido: instale o Docker Desktop e tente de novo.
- Porta 8080 ocupada: feche a outra aplicação e abra novamente.

## Observação
O instalador cria a estrutura em `C:\HopeSoft` e monta o atalho apontando para `INICIAR_HOPESOFT.bat`.
