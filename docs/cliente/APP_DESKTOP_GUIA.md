# 🖥️ HOPESOFT - App Desktop (SEM Terminal)

Este guia é o jeito mais simples de o cliente usar o HopeSoft como um “app”.

## O que fazer (recomendado)
1. **Instalar (cria atalho no Desktop)**  
   Clique 2x em `INSTALAR_HOPESOFT.bat`
2. **Iniciar o sistema (sem janela de terminal)**  
   Use o atalho **HopeSoft** ou clique 2x em `HopeSoft-App.bat`
3. **Parar**  
   - Feche o navegador (para parar o app “sem terminal”) **ou**
   - Execute `PARAR_HOPESOFT.bat`

## Acessar o sistema
Abra no navegador:
`http://localhost:8080/index.html`

Na primeira vez, aguarde **2 a 5 minutos** (Docker inicializando).

## Troubleshooting rápido
- **Docker não está rodando:** abra o *Docker Desktop* e tente novamente.
- **Porta 8080 em uso:** feche o programa que está usando a porta e reabra.
