HOPESOFT - Scripts Windows
==========================

Arquivos mantidos nesta pasta:
- INSTALAR_HOPESOFT.bat   Copia o projeto para C:\HopeSoft e cria um atalho na area de trabalho
- INICIAR_HOPESOFT.bat    Sobe os containers em background e abre o sistema
- PARAR_HOPESOFT.bat      Derruba os containers com seguranca
- HopeSoft-App.bat        Launcher silencioso para uso via atalho

Fluxo recomendado:
1. Execute INSTALAR_HOPESOFT.bat
2. Use o atalho "HopeSoft" no Desktop
3. Para encerrar a operacao, use PARAR_HOPESOFT.bat
